# HHグループ（阪急阪神HD）AI経営診断PoC ｜ モックアプリ

経営判断サイクル（気づく → 見極める → 打ち手を選ぶ）を、2レイヤーで提示するGradioモック。

> PoC前提：簡素UI・**精度保証なし**・公開／マスキングデータ前提。表示値は**すべて架空のダミー**です。

## 構成（2レイヤー）
- **定型レイヤー（計器盤）**… 毎回同じ固定フォーマット。ルール＋固定スコアで決定的に算出。
  - 経営サマリ（結論ファースト：重要度スコア上位の論点カード）
  - モニタリング（計画vs実績／前提vs外部／報告変化点／議論カバレッジ）
  - 診断（重要度／感応度／波及構造／根拠トレース）
- **非定型レイヤー（AIアシスタント）**… OpenAI `gpt-5-mini`。全社ダミーデータを参照したRAG型で、
  打ち手の網羅生成・壁打ち・意思決定ログ。回答には出典を併記。
- 経営サマリの「深掘り」→ 論点をプリフィルしてAIアシスタントへ遷移。

## ファイル
| ファイル | 役割 |
|---|---|
| `app.py` | 画面の組み立て（Gradio）。UI・グラフ・タブ。 |
| `data.py` | ダミーデータと算出ロジック（論点モデル・スコア・RAGコーパス）。 |
| `assistant.py` | AIアシスタントの応答（OpenAI呼び出し）。 |
| `.env` | `OPENAI_API_KEY` / `OPENAI_MODEL`（**非コミット**）。 |

## デザイン
阪急阪神グループの伝統色「**阪急マルーン**」を差し色に、ライト×アイボリー基調。簡素・余白主義。

## 起動
```bash
cd mock_app
pip install -r requirements.txt   # 初回のみ
python3 app.py                    # → http://127.0.0.1:7860
```
必要: Python 3.11+ / gradio 6 / pandas / plotly / openai / python-dotenv

## 注意
- `.env` のAPIキーは秘密情報。Git等で共有しないこと。チャットに貼った場合は**ローテーション（再発行）推奨**。
